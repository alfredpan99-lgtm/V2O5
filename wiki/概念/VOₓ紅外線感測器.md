# VOₓ 紅外線感測器

## 核心定義

VOₓ（非化學計量釩氧化物，x ≈ 1.7–2.0）薄膜是**非冷卻式微測輻射熱計（microbolometer）**的主流感測材料。其電阻對溫度變化高度敏感，可在室溫下偵測長波紅外輻射（LWIR，8–14 μm），無需液氮冷卻。

---

## 重要細節

### 電阻溫度係數（TCR）

$$TCR = \frac{1}{R}\frac{dR}{dT} \approx -2\% \sim -3\% \ /°C$$

TCR 絕對值越大，同樣溫度變化產生的電阻變化越大，感測器越靈敏。

### 材料相比較

| 材料 | TCR | 備註 |
|------|-----|------|
| 純 V₂O₅ | −2% / °C | 電阻率偏高 |
| VO₂ | 相變附近可達 −10% / °C | 相變溫度 68°C，操作範圍受限 |
| **VOₓ（混相，最實用）** | −2% ~ −3% / °C | TCR 與電阻率可透過沉積條件調控 |
| 非晶矽（a-Si） | −2% ~ −3% / °C | 製程更簡單，TCR 略低 |
| YBCO | −3% / °C | 製程複雜 |
| Pt、Ti | −0.2% / °C | 需冷卻，靈敏度低 |

### 工作原理

```
入射紅外輻射
      ↓
吸收層（VOₓ 薄膜）升溫 ΔT
      ↓
電阻變化 ΔR = TCR × ΔT × R
      ↓
讀出電路（ROIC）偵測電流變化
      ↓
輸出熱影像
```

每個像素為懸空微橋結構，熱隔離越好 → ΔT 越大 → 靈敏度越高。

### 薄膜製備方法

| 方法 | 特點 |
|------|------|
| 反應式磁控濺射 | 最主流；V 靶在 O₂/Ar 氣氛中濺射，精確控制 x 值 |
| 脈衝雷射沉積（PLD） | 薄膜品質高，適合研究用途 |
| 溶膠凝膠（Sol-gel） | 低成本，均勻性較差 |
| ALD | 超薄膜與共形覆蓋，新興方向 |

### 關鍵性能指標

- **TCR**：目標 > |2%| / °C
- **電阻率**：1–10 Ω·cm
- **1/f 雜訊**：主要雜訊來源，與薄膜晶體缺陷密度相關
- **NETD（等效雜訊溫差）**：商用產品目標 < 50 mK

---

## 應用場景

- **軍用熱像儀**：夜視、目標識別
- **工業熱成像**：設備預測性維護、電路板檢測
- **醫療**：體溫篩查、血流成像
- **消費性電子**：智慧型手機熱像模組（如 FLIR Lepton）
- **自駕車**：惡劣天候下的行人偵測

### 代表性商用產品

- **FLIR Lepton / Boson**：VOₓ 微測輻射熱計，像素尺寸 12–17 μm
- **DRS / BAE Systems**：1990 年代最早商業化 VOₓ IRFPA 的廠商
- 解析度範圍：160×120 至 1280×1024
