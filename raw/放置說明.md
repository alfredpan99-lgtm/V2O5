# raw/ 放置說明

這個資料夾是所有原始資料的暫存區。

## 什麼東西放這裡

- 用 Obsidian Web Clipper 擷取的網頁文章（.md 格式）
- arXiv 論文的全文或摘要
- GitHub repo 的 README 或技術文件
- 你想消化的任何文字資料

## 放進來之後怎麼做

1. 打開 `系統/編譯提示詞.md`
2. 複製提示詞範本
3. 將此資料夾中的文件內容一起貼給 LLM（例如 Claude）
4. 將 LLM 輸出的內容貼入對應的 `wiki/概念/` 檔案
5. 更新 `wiki/索引/總索引.md`

## 命名建議

```
YYYY-MM-DD_來源_標題.md
```

例如：
```
2026-06-09_arXiv_Attention_Is_All_You_Need.md
2026-06-09_WebClipper_Karpathy-LLM知識庫架構.md
```

## 處理完的文件

原始文件可以保留在這裡作為參考，也可以移至 `raw/已處理/`。
