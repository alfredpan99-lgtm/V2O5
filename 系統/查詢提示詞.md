# 查詢提示詞

用於第三階段：向 LLM 提出複雜研究問題，答案自動歸檔回 wiki。

---

## 提示詞範本

```
你是一個研究助理，可以存取以下知識庫文章。

## 知識庫內容
[貼上 wiki/概念/ 中相關文章的內容]

## 我的問題
[在此輸入你的研究問題]

## 輸出要求
請以 Markdown 格式回答，結構如下：
- `# 問題`：重述問題
- `## 回答`：完整回答
- `## 關鍵洞見`：2-3 個最重要的發現，條列式
- `## 延伸問題`：基於此回答，建議 3 個值得進一步探索的問題
- `## 相關概念`：用 [[雙括號]] 列出此回答涉及的概念

回答完成後，此檔案可直接存入 wiki/衍生/查詢回答/。
```

---

## 儲存規則

查詢回答檔案命名格式：
```
wiki/衍生/查詢回答/YYYY-MM-DD_問題關鍵字.md
```

例如：
```
wiki/衍生/查詢回答/2026-06-09_Transformer注意力機制比較.md
```
